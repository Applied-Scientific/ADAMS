# TXT Scenario Configs (Automation-Friendly)

Flat `key=value` config files used by the TXT-contract prompts.

Rules:
- Paths are repository-relative.
- Booleans use `true|false`, null uses `null`.
- Lists are comma-separated (e.g., `docking_centers=x,y,z`).
