# Testkit Inputs

Scenario input files for the BRD4 test scenario.

Files:
- `protein_waters_apo_brd4.pdb` — BRD4 docking receptor (protonated apo structure with conserved waters)
- `ligands_brd4.csv` — BRD4 ligand set (SMILES)
- `xray_map_brd4.txt` / `xray_map_brd4.yaml` — X-ray reference map for redocking validation
- `3mxf.cif`, `3svg.cif`, `3u5j.cif`, `3u5l.cif`, `4hbv.cif` — BRD4 crystal structures (redocking references)
